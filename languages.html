Programming Languages

I love learning new programming languages and frameworks.

Whenever I learn a new language I typically start with creating a chessbot or a rubik's cube solver.

Java

Java is the first language I would say I truly learned at a deep level.

It's known for its object-oriented nature and its verbosity, with ridiculously long class names like "AbstractSingletonProxyFactoryBean".

The Spring framework is my go-to for for building applications out of any language/framework combo.

Python

I taught myself python in 6th grade when I was stuck in my room for a week with the flu.

Python is my go-to whenever I need to quickly process some data or update some files.

While easy to write, my python code tends to devolve to a spaghetti-code mess when troubleshooting.

Go

Co-developed by Ken Thompson (creator of a little-known language called C) while working at Google.

Go's lightweight virtual threads (known as GoRoutines) make writing highly-concurrent code a breeze.

Popular tools written in go are:

  • Docker
  • Kubernetes
  • Terraform
